# 🔍 Search App (Unsplash Image Search)

This project is a simple **image search application** that fetches and displays images based on a user’s search query using the **Unsplash API**.

Users can search for images, view the results dynamically, and clear the search results when needed.

---

## 🚀 Features
- Image search using Unsplash API
- Dynamic image rendering based on search keywords
- Clear search results functionality
- DOM manipulation with Vanilla JavaScript

---

## 🛠 Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Fetch API
- Unsplash API
